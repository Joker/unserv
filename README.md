# UnServ
API stub server with reverse proxy

```
Usage of ./unserv:
  -js
    	write setupProxy.js file
  -log
    	show request log
  -p int
    	server port (default 8080)
  -proxy string
    	proxy url (example:  http://localhost:9000 )
  -reread
    	disable endpoint file cache
  -root string
    	app files root path (default "./build")
```
